# Email_validation
A simple web application that validates email addresses in real-time using HTML, CSS, and JavaScript.

## 🚀 Features

- Real-time email validation
- Clean and responsive UI
- Feedback on valid/invalid emails

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
